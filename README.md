# Flight Price Tracker

Telegram-бот для отслеживания цен на авиабилеты.

## Планируемый функционал

- Отслеживание стоимости авиабилетов;
- Уведомления об изменении цены;
- История цен;
- Поддержка нескольких пользователей;
- Работа с несколькими источниками данных (провайдерами).

## Структура базы данных

Для хранения информации о пользователях, треках, источниками данных и истории цен используется реляционная база данных.
